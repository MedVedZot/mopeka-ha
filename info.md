Home Assistant integration for Mopeka ultrasonic propane tank sensors via Mopeka Cloud API.

## Features

- Monitor propane tank levels in real-time
- Works with Mopeka WiFi gateways
- Automatic sensor discovery
- Historical data tracking

## Requirements

- Home Assistant 2024.1.0 or newer
- Mopeka account with WiFi-connected sensors
- mopeka-api==1.0.2

## Installation

1. Add this repository as a Custom Repository in HACS
2. Category: Integration
3. Install and restart Home Assistant
4. Add integration with your Mopeka credentials
